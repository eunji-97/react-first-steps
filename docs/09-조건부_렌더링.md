# 조건부 렌더링
어떠한 조건에 의해 렌더링이 달라지는 것을 의미한다.

### 예시
```js
    function Greeting(props) {
        const isLoggedIn = props.isLoggedIn;
        if(isLoggedIn) {
            return <UserGreeting />
        }

        return <GuestGreeting />
    }
```

## 인라인 조건
if 문과 동일한 효과를 내기 위해 && 연산자를 사용한다.

```js
    <h1>안녕하세요</h1>
    {unreadMessages.length > 0 && // if 문의 and 연산자
        <h2>
            현재 읽지 않은 메시지가 {unreadMessages.length}개 있습니다.
        </h2>
    }
```
개수에 따라 뒤에 나오는 태그가 렌더링이 되거나 안되거나 함

```js
    function Counter(props) {
        const count = 0;

        return (
            <div>
                {count && <h1>현재 카운트 : {count}</h1>}
            </div>
        )
    }
```
조건 count 가 0이므로 false라 count 값인 0이 그대로 표시 '<div>0</div>' 이 된다.

if-else는 삼항 연산자를 사용한다.

## 컴포넌트 렌더링 막기
리액트에서는 렌더링 하고 싶지 않을 떄, null 을 반환하여 막아야 한다.

```js
    function WarningBanner(props){
        if(!props.warning) {
            return null;
        }

        return (
            <div>경고!</div>
        )
    }
```
