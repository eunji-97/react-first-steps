# 엘리먼트
브라우저에서 보이는 HTML 요소 > DOM 엘리먼트

## 리액트 엘리먼트
리액트의 virtual DOM에 존재하는 엘리먼트 
createElement()가 리엑트 엘리먼트를 생성하고, 리액트가 이를 이용해 DOM 엘리먼트를 생성한다.

## 생김새
리액트 엘리먼트는 자바스크립트 객체 형태로 존재한다.
`{` 
`   type : 'button',`
`   props : {`
`       className : 'bg-green', `
`       children : {`
`           props : { chlidren : 'Hello!'}`
`   }`
`}`

type은 태그나 리액트 컴포넌트, props는 속성값이 된다.

### 랜더링 시 (DOM 엘리먼트가 된다.)
`<button class='bg-green'>` 
`   <b>`
`       Hello!`
`   </b>`
`</button>`

## 특징
불변성 : 엘리먼트가 생성된 후에는 바뀔 수 없다.
변경된 엘리먼트를 보여주려면? > 새로운 엘리먼트를 만들어 바꿔줘야 한다.

## 랜더링
`<div id="root"></div>` root DOM Node로, virtual DOM에서 가장 최상단에 위치해있다.

이 div에 랜더링 하기 위해서는,
`const element = <h1>안녕!</h1>`
`const root = ReactDOM.createElement(document.getElementById('root'));`
`root.render(element);`
render 함수를 이용한다.
