# 합성 (composition)
여러 개의 컴포넌트를 합쳐 새로운 컴포넌트를 만드는 것을 의미한다.

## 합성 사용 기법
### Containment
하위 컴포넌트를 포함하는 형태의 합성 방법.  

하위 컴포넌트가 어떤 것이 오는지 예상할 수 없을 때 사용한다.

```js
function FancyBorder(props) {
    return (
        <div className={'FancyBorder FancyBorder-' + props.color}>
            {props.children}
        </div>
    );
}

function WelcomeDialog() {
    return (
        <FancyBorder color="blue">
            <h1 className="Dialog-title">Welcome</h1>
            <p className="Dialog-message">Thanks for visiting our site!</p>
        </FancyBorder>
    );
}
```
children에 h1, p 태그가 props로 전달된다.

### Specialization
범용적인 컴포넌트를 만들어 놓고 이를 특수화 시켜서 컴포넌트를 사용하는 방식이다.

```js
function Dialog(props) {
    return (
        <div className="Dialog">
            <h1 className="Dialog-title">{props.title}</h1>
            <p className="Dialog-message">{props.message}</p>
        </div>
    );
}

function ConfirmDialog() {
    return <Dialog title="Confirm" message="Are you sure?" />;
}
```
