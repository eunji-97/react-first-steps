# 리스트 / 키
리스트는 배열 구조, 키는 각 객체나 아이템을 구분할 수 있는 고유한 값을 의미한다.

## 여러 개의 컴포넌트 렌더링
배열의 map 함수를 이용해 배열에 들어있는 각 변수에 처리를 한 후 리턴하는 방식으로 렌더링한다.

```js
    const doubled = numbers.map((number) => {number * 2}) //map 사용 예제

    const numbers = [1, 2, 3, 4, 5];
    const listItems = numbers.map((number) => 
        <li>{number}</li> //리스트 출력을 할 수 있다.
    )
```

## 리스트의 키
리액트에서의 키는 같은 리스트에 있는 엘리먼트 사이에서만 고유한 값이면 된다.
(그리고 꼭 필요하다.)

### 키값으로 id 사용 (고유한 id가 있는 경우)
```js
    const todoItems = todos.map((todo) => 
        <li key={todo.id}>
            {todo.text}
        </li>
    )
```

### 키값으로 인덱스 사용 (고유한 id가 없는 경우)
```js
    const todoItems = todos.map((todo, index) => 
        <li key={index}>
            {todo.text}
        </li>
    )
```
